# pbixOptions.md

## #1 Persist Hierarchy Level
**Location:** Options > Current File > Report settings > Field Parameter Options > Persist Hierarchy Level  

Controls whether Matrix visuals retain their expanded or collapsed hierarchy state when a field parameter changes.  
- **On:** keeps the current expansion.  
- **Off:** collapses to the top level on change.
